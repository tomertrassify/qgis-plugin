# Grid Quick GeoJSON Export

QGIS-Plugin fuer schnellen Einzel-Export von Features als GeoJSON direkt aus dem Rechtsklick-Menue auf der Karte.

## Funktion

- Rechtsklick auf ein Feature in der Karte.
- Menueeintrag `Schnellexport GeoJSON` auswaehlen.
- Speicherort waehlen.
- Das geklickte Feature wird als einzelne GeoJSON-Datei exportiert.

## Installation

- ZIP-Datei in QGIS ueber `Plugins > Plugins verwalten und installieren... > Aus ZIP installieren` laden.
